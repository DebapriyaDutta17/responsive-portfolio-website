# Responsive Portfolio Website

## 🌟 Overview

A modern, fully responsive personal portfolio website developed using HTML, CSS, and JavaScript. The website showcases personal information, skills, projects, education, and contact details in a professional and visually appealing manner.

The portfolio is designed to work seamlessly across desktops, tablets, and mobile devices.
